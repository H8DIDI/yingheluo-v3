🎆 萤合落 — 专业烟花燃放控制与3D编排系统

萤合落（Yingheluo）是一款基于 React 19 + Three.js 的专业烟花燃放控制与编排系统， 提供3D实时烟花模拟预览、多阵地管理、AI智能编排等功能。适用于烟花秀设计、花炮产业数字化和燃放培训。

✨ 核心功能

  • 3D实时烟花模拟预览 — 50000粒子物理引擎，真实弹道模拟与金属盐冷却发色模型
  • 800管专业阵地布局 — 前线弧形、双翼包围、中央矩阵、后排高空多层立体编排
  • 10+种烟花效果 — 牡丹、菊花、柳、彗星、十字星、地雷、喷泉等，各有独立物理特性
  • AI智能编排助手 — 自然语言描述需求，AI自动生成专业烟花秀方案
  • 时间轴波次编辑 — 可视化拖拽编排，支持齐射、波浪、螺旋等发射模式
  • 多阵地对称齐射 — 左右翼同步、中心轴对称、V字波浪等专业编排模式
  • 完整的烟花资产管理 — 数据库管理烟花效果库，支持自定义效果创建

🏗️ 技术架构

前端：React 19 + TypeScript + Vite 6 + Three.js + Zustand 状态管理
3D引擎：Three.js InstancedMesh + 自研物理引擎（二次方阻力模型）
后端：Cloudflare Workers + D1 数据库
部署：Cloudflare Pages（全球CDN加速）

🎇 浏阳花炮 — 世界烟花之乡

萤合落系统的设计灵感来源于中国湖南浏阳——世界著名的烟花之乡。 浏阳花炮产业拥有超过1400年的历史，是全球最大的烟花生产基地。

浏阳花炮产业年产值达508.9亿元，拥有431家规模以上企业和3721项烟花相关专利。 在文化创意赋能背景下，浏阳正从传统花炮制造向数字化、智能化转型， 萤合落正是服务于这一转型的数字化工具——让烟花秀设计从经验驱动走向数据驱动。

从北京奥运会到国庆70周年，从浏阳国际烟花大赛到全球各地的"天空剧院"， 专业烟花编排系统正成为花炮产业价值提升的关键路径。
