## 🔬 烟花的化学原理

烟花是化学能向光能、热能和动能转化的艺术。每一朵绽放的烟花背后，都是一场精密的氧化还原反应。

### 基础燃烧反应

**黑火药** —— 烟花延续千年的推进剂，经典配比 75:15:10（硝酸钾:木炭:硫磺）：

$$2\text{KNO}_3 + 3\text{C} + \text{S} \rightarrow \text{K}_2\text{S} + \text{N}_2\uparrow + 3\text{CO}_2\uparrow$$

| 组分 | 化学式 | 角色 |
|:---:|:---:|:---|
| 硝酸钾 | KNO₃ | 氧化剂 — 提供反应所需的氧 |
| 木炭 | C | 还原剂/燃料 — 与氧剧烈反应 |
| 硫磺 | S | 助燃剂 — 降低着火点，加速反应 |

### 金属盐发色原理

烟花的绚丽色彩源于 **金属离子的焰色反应** —— 高温下金属原子电子跃迁至激发态，回落基态时以特定波长的可见光释放能量：

| 颜色 | 金属元素 | 常用化合物 | 发射波长 |
|:---:|:---:|:---:|:---:|
| 🔴 红色 | 锶 (Sr) | SrCO₃、Sr(NO₃)₂ | 606–636 nm |
| 🟢 绿色 | 钡 (Ba) | BaCO₃、Ba(NO₃)₂ | 505–535 nm |
| 🔵 蓝色 | 铜 (Cu) | CuCO₃、CuCl₂ | 420–460 nm |
| 🟡 黄色 | 钠 (Na) | NaNO₃、NaHCO₃ | 589 nm（D线）|
| ⚪ 白/银 | 铝/镁/钛 | Al、Mg、Ti 粉 | 宽谱白光 |
| 🟠 橙色 | 钙 (Ca) | CaCO₃、CaCl₂ | 590–620 nm |
| 💜 紫色 | 锶 + 铜 | SrCO₃ + CuCO₃ | 红蓝混合 |
| 🥇 金色 | 铁/碳 | Fe 粉、木炭颗粒 | 连续谱金色火花 |

> **蓝色是最难实现的颜色。** CuCl 分子在高温下极易分解，温度必须精确控制在 1000–1200°C 之间——过高蓝色消失，过低反应不充分。这是烟花行业公认的"蓝色之难"。

### 关键化学方程式

**亮珠（星体）中的典型反应：**

红色亮珠的发色机制 — PVC 作为氯供体与金属氧化物反应生成金属氯化物：

$$\text{KClO}_4 + 2\text{Mg} \rightarrow \text{KCl} + 2\text{MgO} + \text{光和热}$$

$$\text{SrCO}_3 \xrightarrow{\Delta} \text{SrO} + \text{CO}_2\uparrow$$

$$\text{SrO} + 2\text{HCl}_{(\text{来自 PVC 热解})} \rightarrow \text{SrCl}_2 \xrightarrow{\text{激发}} \text{红色光}$$

**金属粉燃烧产生闪光和拖尾效果：**

$$4\text{Al} + 3\text{O}_2 \rightarrow 2\text{Al}_2\text{O}_3 + \text{强白光（银白闪光）}$$

$$2\text{Mg} + \text{O}_2 \rightarrow 2\text{MgO} + \text{强白光}$$

$$\text{Ti} + \text{O}_2 \rightarrow \text{TiO}_2 + \text{银色火花拉尾}$$

### 前沿：绿色烟花化学

传统烟花的环保痛点在于 SO₂ 排放和重金属残留。2024 年浏阳研发的 **W49 无硫复合还原剂**，用金属合金粉和有机还原剂替代硫磺，实现了 SO₂ 零排放。2025 年亚冬会开幕式全部采用微烟无硫产品，标志着绿色烟花从实验室走向大型实战。

## 🏭 烟花制造工艺

> ⚠️ **安全声明**：以下内容仅供科普教育。烟花制造属于国家严格管控的高危行业，需持有相关资质并遵守《烟花爆竹安全管理条例》等法律法规。**严禁未经许可私自制造烟花爆竹。**

### 一、原料配制（配药）

配药是烟花制造中 **最危险也最关键** 的环节，每一种组分的比例直接决定产品的安全性和效果。

| 组分 | 作用 | 常用材料 | 注意事项 |
|:---:|:---|:---|:---|
| 氧化剂 | 提供氧气支持燃烧 | KClO₄（高氯酸钾）、KNO₃、Ba(NO₃)₂ | 与还原剂隔离储存 |
| 还原剂 | 燃烧产生高温和光 | Al 粉、Mg 粉、Ti 粉、Fe 粉 | 粒度决定燃速 |
| 发色剂 | 产生特定颜色 | SrCO₃（红）、BaCO₃（绿）、CuCO₃（蓝） | 含量影响色纯度 |
| 粘合剂 | 成型固化 | 虫胶（紫胶片）、糊精、酚醛树脂 | 过多影响燃烧 |
| 氯供体 | 增强金属焰色 | PVC 粉、六氯苯、氯化石蜡 | 现代配色核心技巧 |
| 缓燃/助燃 | 调节燃速 | 硫磺（助燃）、碳酸钙（缓燃） | 精确控制比例 |

**配药精度**：各组分称量误差控制在 **±0.5%** 以内。浏阳现有 20 条自动化示范线，关键涉药工序自动化率达 80%。

### 二、造粒（制作亮珠/星体）

亮珠是烟花绽放时散射出的发光颗粒，是烟花效果的基本单元。清代浏阳名匠李熙雅（"李四美"坊坊主）以精湛的亮珠技艺著称，其大叶兰花、大叶菊花、连升三级等品种流传至今。

**工艺流程：**

```
混药 ──→ 润湿 ──→ 滚制/切割 ──→ 烘干
 │          │          │            │
 ↓          ↓          ↓            ↓
防爆搅拌   加酒精/    造粒锅滚珠   40-60°C
均匀混合   粘合剂     或压片切割   缓慢烘干
```

**分层亮珠（变色弹核心技术）：** 在造粒锅中先滚制核心层，烘干后再包裹外层药剂，实现颜色渐变效果。例如内红外绿——绽放时先看到绿色，燃至内层变红色。

### 三、装药与压实 —— "药实则缓，药松则炸"

装药工艺是决定烟花安全性与燃烧特性的 **最关键步骤**。

#### 密度与燃速的核心关系

```
┌─────────────────────────────────────────────┐
│  密度高 → 颗粒间空隙小 → 火焰传播面积小     │
│         → 燃烧慢而稳定 → ✅ 安全可控         │
│                                             │
│  密度低 → 颗粒间空隙大 → 氧气充分渗透       │
│         → 全面同时着火 → ❌ 爆炸             │
└─────────────────────────────────────────────┘
```

- 🟢 **压实良好**：药剂致密无缝，燃烧面沿表面逐层推进，速度可控
- 🔴 **压实不足（存在空隙）**：火焰钻入缝隙，表面积骤增，瞬间剧烈燃烧 → **爆燃或爆炸**
- 🟡 **过度压实**：颗粒被压碎改变粒度分布，或摩擦生热引发危险

#### 震动棒（振动压实技术）

工业生产中使用 **震动压实设备（震动棒/振动台）** 确保药剂均匀致密：

- 药剂装入纸筒后，高频振动使颗粒紧密排列
- 消除内部气泡和空隙
- 频率、振幅和时间针对不同药剂精确调控
- **目标密度**：达到理论密度的 85%–95%

> 💡 2019 年浏阳"12·4"碧溪烟花厂事故（13 人死亡）的直接原因就是工人搬运药饼时抛甩动作导致摩擦撞击引发爆炸。这一惨痛教训深刻说明了装药环节安全管控的极端重要性。事故后浏阳全面推行涉药工序"单人单栋"作业制度和蓝/黄/橙/红四色工房分级管理。

### 四、组装结构

**礼花弹的内部结构：**

```
         ┌── 引线（导火索）── 延时精度 ±0.3s
         │
    ┌────┴────┐
    │ 开爆药  │ ← 中心装药（黑火药/闪光药）
    │（中心） │   点燃后产生高温高压气体
    └────┬────┘
    ·····│·····
    · 亮珠层 · ← 效果药（星体），排列方式决定绽放形状
    · ········ ·   球形排列→球形绽放 | 环形排列→环形效果
    ·····│·····
    ┌────┴────┐
    │ 开包药  │ ← 推动亮珠向四周扩散，用量决定绽放直径
    └────┬────┘
    ┌────┴────┐
    │  外壳   │ ← 纸壳/塑壳，承受发射膛压
    └─────────┘
```

## 🎨 烟花产品分类与燃放效果

烟花按国标 GB 10631 分为 A、B、C、D 四级。2026 年即将实施的新国标 GB 10631-2025 整合 7 项旧标准，标志着行业走向精细化规范。

### 一、小型产品（C/D 级 · 个人燃放）

| 产品 | 效果 | 典型规格 |
|:---:|:---|:---:|
| 吐珠 | 连续吐出彩色火球，伴有啸声 | 6–30 发 |
| 喷花 | 锥形/柱形喷射，扇形火花帘 | 高 2–8m |
| 旋转 | 地面或空中高速旋转，彩色光环 | 地旋/天旋 |
| 线香 | 手持发光棒，金色火花 | 30–120s |
| 小礼花 | 升空后小范围绽放 | 直径 1–3m |

近年来浏阳企业推出戏曲脸谱、华夏图腾、二十四节气等文创烟花，"加特林"等网红产品毛利率可达 100% 以上。

### 二、组合烟花（B/C 级 · 中型燃放）

| 类型 | 规格 | 口径 |
|:---:|:---|:---:|
| 小型组合 | 25–36 发，低空绽放 | 20–25mm |
| 中型组合 | 49–100 发，色彩丰富 | 25–30mm |
| 大型组合 | 100–500 发，高空绽放 | 30–50mm |

**经典效果：** 🌸 牡丹（球形丰满）· 🌼 菊花（细长拖尾）· 🌿 柳（垂柳飘逸）· 👑 锦冠（密集闪耀）· 🌴 椰子树 · 💧 瀑布

### 三、礼花弹（A/B 级 · 专业燃放）

浏阳花炮在全国专业燃放市场份额超过 **90%**。

| 口径 | 升空高度 | 绽放直径 |
|:---:|:---:|:---:|
| 2 寸 (50mm) | ~60m | ~30m |
| 3 寸 (75mm) | ~100m | ~60m |
| 6 寸 (150mm) | ~180m | ~150m |
| 8 寸 (200mm) | ~220m | ~200m |
| 12 寸 (300mm) | ~300m | ~300m |

**高端效果**：变色弹 · 多重芯弹（环中环）· 图案弹（笑脸/心形/五角星）· 锦冠菊（二次绽放）

| 特效 | 描述 |

## 🚀 快速开始

### 环境要求

- Node.js 20+（推荐 22）
- npm 或 pnpm

### 本地开发

```bash
git clone https://github.com/H8DIDI/yingheluo-v3.git
cd yingheluo-v3
npm install
npm run dev
```

访问 `http://localhost:5173`。局域网访问用 `npm run dev:lan`。

### AI 助手配置

1. 进入 AI 助手页面 → 点击 **⚙️**
2. 填写 API 地址、密钥、模型 → 保存
3. 输入 `生成方案 + 主题描述` 即可生成燃放方案

支持 [OpenRouter](https://openrouter.ai/)（有免费模型）、OpenAI、本地模型等任何兼容接口。

## ☁️ 部署指南

### Cloudflare Pages（推荐）

1. Fork 本项目 → 登录 [Cloudflare Dashboard](https://dash.cloudflare.com/) → Pages → 创建项目
2. 连接 GitHub，构建命令 `npm run build`，输出目录 `dist`
3. 创建 D1 数据库并执行迁移：

```bash
wrangler d1 create yingheluo-v3
wrangler d1 execute yingheluo-v3 --file=migrations/0001_init.sql
wrangler d1 execute yingheluo-v3 --file=migrations/0002_create_chat_log.sql
```

4. 修改 `wrangler.toml` 中的 `database_id` → 部署 → 绑定域名

### 环境变量（可选）

| 变量名 | 说明 | 默认值 |
|:---:|:---|:---:|
| `VITE_AI_API_URL` | AI API 地址 | `https://openrouter.ai/api/v1` |
| `VITE_AI_API_KEY` | AI API 密钥 | (空) |
| `VITE_AI_MODEL` | AI 模型 | `xiaomi/mimo-v2-flash:free` |
