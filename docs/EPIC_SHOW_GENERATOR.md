# 史诗级烟花秀生成器

## 概述

这是一个专业的烟花秀算法生成器，能够自动创建一场**10分钟（600秒）**的史诗级烟花表演。生成器严格遵循专业烟花秀的起承转合结构，并实现了排浪式齐射逻辑，避免了"撒尿式"的单发效果。

## 核心特性

### 1. 起承转合结构

烟花秀被精心划分为四个阶段：

#### 🎆 开场爆发（0-60秒）
- **特点**：密集、快节奏、震撼开场
- **颜色**：金色、银色、白色为主
- **高度**：80-100米（较低空）
- **节奏**：每1-2秒一波
- **效果**：burst（爆裂）、peony（牡丹）、comet（彗星）

#### 🎨 主题章节（60-480秒）
分为6个不同颜色的主题章节，每个章节约70秒：

1. **红色激情**（60-130s）- 热烈、激昂
2. **蓝色海洋**（130-200s）- 宁静、深邃
3. **紫色梦幻**（200-270s）- 神秘、浪漫
4. **绿色生机**（270-340s）- 清新、活力
5. **橙色温暖**（340-410s）- 温馨、柔和
6. **彩虹交响**（410-480s）- 多彩、和谐

- **节奏**：舒缓有致，每2.5-4秒一波
- **高度**：100-140米（中高空）
- **效果**：peony（牡丹）、willow（柳树）、crossette（十字星）、burst（爆裂）

#### 🚀 前终曲（480-540秒）
- **特点**：节奏加快，多层级配合
- **策略**：低空（80-110米）+ 高空（130-160米）交替
- **节奏**：每1.5-2.5秒一波
- **配合**：70%的波次会在0.5秒后触发高空波次

#### 💥 终极高潮（540-600秒）
- **特点**：超密集齐射，满屏覆盖
- **节奏**：每0.5-1秒一波
- **策略**：多个阵地同时发射（2-4个）
- **最后5秒**：全场所有炮筒齐射，每0.3秒一轮
- **高度**：100-160米（全高度范围）
- **颜色**：所有颜色混合

### 2. 排浪式齐射逻辑

**核心原则**：绝对禁止随意单发！

每个波次（Wave）包含：
- **5-10个炮筒**同时发射
- **完全相同的发射时间**（launchTime）
- **完全相同的目标高度**（targetHeight）
- **相同的颜色和效果类型**

这确保了���花在空中形成整齐的排浪效果，而不是零散的"撒尿式"发射。

### 3. 高度安全控制

所有生成的烟花高度都经过安全限制：

```typescript
safeHeight = Math.min(randomHeight, MAX_SCENE_HEIGHT * 0.8)
```

- 场景最大高度：200米
- 安全高度限制：160米（80%）
- 确保烟花在半空中爆炸，不会撞击顶部边界

### 4. 智能弹药管理

生成器实现了自动换弹逻辑：

- 跟踪每个炮筒的装填和发射状态
- 当可用炮筒不足时，自动重置所有炮筒（模拟换弹）
- 确保两次发射间隔大于5秒（通过波次时间控制）

## 技术细节

### 数据结构

每个波次（Wave）包含：

```typescript
interface Wave {
  time: number;              // 发射时间（秒）
  positionId: string;        // 阵地ID
  rackId: string;            // 炮架ID
  tubeIndices: number[];     // 炮筒索引数组
  height: number;            // 目标高度（米）
  color: string;             // 颜色（十六进制）
  effectType: FireworkType;  // 效果类型
  pattern: FiringPattern;    // 发射模式
}
```

### 发射模式

- **all**：所有炮筒同时发射（最常用）
- **sequential**：从左到右依次发射（间隔100ms）
- **wave**：波浪式发射（用于矩阵）

### 烟花效果类型

- **peony**（牡丹）：经典球形爆炸，粒子数150
- **burst**（爆裂）：强烈爆炸，强度1.0
- **willow**（柳树）：下垂效果，持续时间3.5秒
- **crossette**（十字星）：分裂效果
- **comet**（彗星）：拖尾效果

## 性能优化

生成器采用了多项优化措施：

1. **算法生成**：不直接输出巨大JSON，而是通过算法动态生成
2. **内存效率**：使用引用而非复制，减少内存占用
3. **类型安全**：完整的TypeScript类型定义
4. **可扩展性**：易于添加新的章节或效果类型

## 自定义扩展

### 添加新的主题章节

在 `generateActs()` 函数中添加新的章节：

```typescript
const acts = [
  // ... 现有章节
  {
    name: '你的主题',
    start: 410,
    end: 480,
    colors: ['#YOUR_COLOR'],
    tempo: 3
  },
];
```

### 调整波次密度

修改时间间隔参数：

```typescript
// 更密集
for (let t = start; t < end; t += random(1, 1.5)) { ... }

// 更稀疏
for (let t = start; t < end; t += random(4, 6)) { ... }
```

### 修改高度范围

调整高度参数：

```typescript
// 更高
const height = random(150, 180);

// 更低
const height = random(60, 90);
```
