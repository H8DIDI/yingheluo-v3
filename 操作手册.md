# 萤合落 V1.0 全中文操作手册

## 1. 环境准备
- Node.js 20+（推荐 22/24）
- Windows PowerShell 或 CMD
- 局域网访问时请确保防火墙放行端口：5173（前端）

## 2. 安装依赖
```bash
npm install
```

## 3. 一键启动（局域网）
双击 `start-lan.bat`，将自动启动前端并映射到局域网。
- 前端地址：`http://本机IP:5173`
- API 由 Cloudflare Pages Functions 提供：`/api`

提示：本机 IP 可通过命令 `ipconfig` 查看。

## 4. 手动启动
```bash
npm run dev:lan
```

如果仅本机访问，可使用：
```bash
npm run dev
```

## 5. 后台管理界面（新增）
入口：工作台右上角「后台管理」按钮。

主要功能：
- 运行状态：显示后端连接情况与数据库路径
- 保存为新工程：将当前工作台工程写入数据库
- 覆盖选中记录：用当前工作台工程覆盖数据库中的指定记录
- 载入工作台：从数据库读取工程到前端工作台
- 删除记录：删除数据库工程记录

## 6. 工作台使用指南
### 6.1 二维地图编辑
- 点击「添加阵地」后在地图放置阵地
- 选中阵地可拖拽移动
- 双击阵地可删除
- 支持网格吸附与缩放

### 6.2 炮架管理
- 选中阵地后可新增炮架
- 支持扇形、直排、矩阵三种炮架类型

### 6.3 时间轴编排
- 点击时间轴空白处切换播放指针
- 拖拽指令块可调整时间与轨道
- 双击指令块可删除
- 可配置燃放模式（齐射、顺序、波浪等）

### 6.4 导出
右上角「导出施工单」可生成 PDF
（同时支持 JSON/CSV 导出接口）

## 7. 数据库说明
- 数据库：Cloudflare D1（绑定名 `DB`）
- 表结构：`projects`
  - `id`：记录 ID
  - `project_id`：工程内部 ID
  - `name`：工程名称
  - `data`：工程 JSON 数据
  - `created_at` / `updated_at`：时间戳

## 8. 后端 API 说明
基础路径：`/api`
- `GET /api/health`：服务健康检查
- `GET /api/projects`：获取工程列表
- `GET /api/projects/:id`：获取工程详情
- `POST /api/projects`：创建工程
- `PUT /api/projects/:id`：更新工程（可覆盖 data）
- `DELETE /api/projects/:id`：删除工程

## 9. 代码结构说明（新增部分）
- 前端后台管理界面：

- Cloudflare 配置：

## 10. 常见问题
- 局域网无法访问：检查防火墙并确保前端使用 `--host 0.0.0.0`
