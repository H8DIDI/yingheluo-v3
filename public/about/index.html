关于天河落 — 浏阳花炮产业聚合站

天河落（Tianheluo）是一款开源的浏阳花炮产业数字化聚合展示站。项目名称取自"天河"——即夜空中壮丽的银河，寓意万千焰火如星河倾泻、璀璨降落。

天河落围绕"视频 · 产业链 · 资讯 · 数据 · 工具"五大板块，让从业者、观察者与爱好者都能以极低成本跟上浏阳花炮产业的最新动态。

🎯 项目背景

中国是烟花的发源地，浏阳作为世界烟花之都，其花炮产业年产值超过508亿元。然而传统的烟花秀设计仍高度依赖人工经验，缺乏可视化编排工具。天河落正是为解决这一痛点而生。

本项目源于对文化创意赋能花炮产业的深度思考——如何用数字技术提升传统产业的价值链？烟花编排系统就是答案之一。

🧩 五大板块

  • 视频墙 — 日更 B 站 / YouTube 浏阳烟花与烟花秀内容
  • 产业链 — 上游原料装备 · 中游药剂组装 · 下游流通应用
  • 资讯 — 百度新闻、财经媒体、政府公众号关键词追踪（标题摘要）
  • 数据 · 企业 — 产业指标 + 规上企业名录（API 对接中）
  • 工具集 — 3D 烟花编排模拟器、燃放时长估算、安全距离计算器等

🏗️ 技术架构

  • 前端：React 19 + TypeScript + Vite 6 + Tailwind + React Router
  • 后端 API：Cloudflare Pages Functions + D1 数据库
  • 数据管线：阿里云 ECS 上 RSSHub + Python 抓取脚本 + systemd timer 日更
  • 分发：Cloudflare 全球 CDN + Cloudflared Tunnel
  • MCP 服务：为 AI 助手提供 list_fireworks_videos / list_fireworks_news / get_industry_stats / get_industry_chain 等工具
  • 3D 引擎：Three.js GPU 粒子 + 金属盐冷却发色模型（工具集内保留）
